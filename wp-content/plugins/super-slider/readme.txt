Plugin that lets you create a slider very easy and with spectacular results.

== Description ==

Multiple configurations and image management simple. You can create multiple sliders and use them on your pages, posts, as a widget or inserting them directly as HTML code. Install the plugin, activate and admin in Settings -> Super Slider.

<a href="http://www.extendyourweb.com/wordpress/super-slider/">Live DEMO and plugin page</a>.

Create the sliders you want, but put just a slider per page. Each slider can manage:


<ul>
<li> Images
<ul>
<li>Url of the image link to the wordpress media library.</li>
<li>title</li>
<li>description</li>
<li>Link</li>
<li>Order</li>
</ul>
</li>
<li> Settings
<ul>
<li>Measures the slider with own measures for mobiles and tablets.</li>
<li>Duration of the transition between images.</li>
<li>Buttons and their position.</li>
<li>Text position and effect.</li>
</ul>
</li>
<li> Effects
<ul>
<li>Effect settings.</li>
<li>Select from over 40 possible transition effects between images.</li>
</ul>
</li>
</ul>


Once installed and activated the plugin go to "Wordpress admin -> settings -> SUPER SLIDER" to create and manage your slider. You can load the slider within a page or post using the code:

`[super 1/] Where 1 is the gallery ID`

You can also use the super slider widget to load the slider into a widget.
		

== Installation ==


1. Upload `super-slider` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure and create new slider in your Wordpress settings: super slider
4. Place [`super` ID /] in your Posts, where ID is the slider ID.

5. Admin params:

Images:

- Url of the image link to the wordpress media library.
- Title
- Description
- Link
- Order

Settings:

- Measures the slider with own measures for mobiles and tablets.
- Duration of the transition between images.
- Buttons and their position.
- Text position and effect.

Effects:

- Effect settings.
- Select from over 40 possible transition effects between images.
